python语法
（1）外部文件调用
可以通过import来引用文件，调用里面的数据
例如：gl.py文件中内容如下
_a = 1
_b = 2
text.py可以通过import gl来引用变量
import gl
def fun():
  print gl._a
  print gl._b
fun()
（2）数组
①元组
是写保护的，即元组创建之后不能再修改
②list
列表和元组相似，也由一组元素组成，列表可以实现添加、删除和查找操作，元素的值可以被修改。可以通过append在尾部添加元素，使用remove删除元素
③字典（类似于Java中的map）
由键-值对组成的集合，字典中的值通过键来引用。键和值之间用冒号隔开，键-值对之间用逗号隔开，并且被包含在一对花括号中。
（3）函数定义
def 函数名（参数）：
    函数内容
    return 返回值   
（4）文件io
f = file("aaa.txt", 'w') // 打开当前文件夹下的为aaa.txt的文件，模式设置为write(类似于Java的fileinputstream)
context = "hello word"
f.write("hello word") or f.write(context)
f.close()
